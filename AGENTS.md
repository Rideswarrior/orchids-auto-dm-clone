## Project Summary
This project is a professional website for EDUSOURCE HRD CENTRE, KOLLAM, focusing on their Hospital Administration Course. It is a transformed version of a cloned landing page, maintaining the original layout and UI while completely replacing the content to suit a professional healthcare education institute.

## Tech Stack
- Framework: Next.js 15 (App Router)
- Styling: Tailwind CSS, CSS Variables
- Icons: Lucide React
- Components: React (Functional Components)

## Architecture
- `src/app/`: Contains the main layout and home page.
- `src/components/sections/`: Modularized UI sections (Hero, Features, FAQ, etc.).
- `src/app/globals.css`: Central styling and theme definitions.

## User Preferences
- Maintain existing layout and structure exactly.
- Focus on professional, realistic, and trust-building content.
- Avoid placement or salary guarantees.
- Use Next.js `Image` component for assets.

## Project Guidelines
- No jobs or placement promises in content.
- Highlight government recognition and professional curriculum.
- Simple and clear English suitable for students and parents.

## Common Patterns
- Modular section components imported into `page.tsx`.
- Use of "use client" for components with state or client-side hooks.
- Styled-jsx or Tailwind for section-specific decorative elements.
